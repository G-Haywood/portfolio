Ossified Orbit

A personal portfolio website built with Astro and Docker and self-hosted on my homelab.
About

This project serves as both my portfolio website and a demonstration of my skills in modern web development, Linux administration, Docker, self-hosting, networking, and cybersecurity.

The website is developed locally on Arch Linux, version controlled with Git, and deployed to my homelab using Docker.
